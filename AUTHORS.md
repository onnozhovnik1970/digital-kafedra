# Цифровий завкаф / Digital Kafedra

## About the Project
AI-powered web application for automating academic staff reporting 
and accreditation monitoring at Ukrainian higher education institutions.
The platform provides personal cabinets for academic staff, 
an AI-assisted admin panel for department heads, and automated 
compliance tracking based on NAZYAVO licensing requirements.

## Key Features
- Personal cabinet for each academic staff member
- AI-powered compliance score calculation
- Automated reporting in PDF and Excel formats
- Admin panel for department heads with AI assistant
- Public profile page for each staff member
- Accreditation commission export tools
